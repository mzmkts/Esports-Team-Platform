# Esports Team Platform (MERN Stack)

A full-stack web application for esports players to create teams, manage profiles, and interact through posts and real-time features.

---
Features

Authentication
- User registration and login
- JWT-based authentication
- Password hashing (bcrypt)
- Protected routes

Teams System
- Create teams
- Join / leave teams
- Team ownership system
- Member management

Posts System
- Create posts
- Like / unlike posts
- View all posts feed
- User-based content

User System
- User profiles
- Update profile (avatar, bio, favorite games)
- Public user profiles
- Team membership display

Real-time (WebSocket)
- Live chat system for teams
- Online users tracking
- Real-time message updates

---

Tech Stack

Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- WebSocket (ws)
- bcrypt

Frontend
- Next.js (App Router)
- JavaScript
- CSS / CSS Modules
